# Spring Boot Project – IntelliJ IDEA 

# Getting Started

### Open the Project in IntelliJ

1. Open IntelliJ IDEA

2. Go to File → Open

3. Select the project folder

4. Click OK

IntelliJ will automatically import the Maven project.

### Wait for Maven Dependencies to Finish Downloading

Look at the bottom-right corner — wait until all background tasks are complete.

### Run the Application

Navigate to the main class: AkantaraHotelApplication

Click the green Run button next to the main method

OR run from terminal: mvn spring-boot:run
